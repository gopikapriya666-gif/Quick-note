# 📝 Quick Note Application

## 📌 Project Overview

Quick Note is a simple note-taking application built using **Python**, **Streamlit**, and **SQLite**. It allows users to write notes, save them to a database, and view all saved notes on the same page.

---

## 🚀 Features

- Create and save notes
- Store notes in SQLite database
- Display all saved notes
- Simple and user-friendly interface
- Fast and lightweight application

---

## 🛠️ Technologies Used

- Python 3
- Streamlit
- SQLite

---

## 📁 Project Structure

```
QuickNote/
│── app.py
│── notes.db
│── requirements.txt
│── README.md
```

---

## ▶️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/QuickNote.git
```

2. Open the project folder

```bash
cd QuickNote
```

3. Install the required package

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python -m streamlit run app.py
```

---

## 📷 Output

- Users can write a note.
- Click **Save Note** to store it.
- Saved notes are displayed below the input area.
